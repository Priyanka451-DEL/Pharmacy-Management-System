# Pharmacy Management System

A Java console-based application to manage medicines in a pharmacy.

## Features
- Add medicine
- View medicine by ID
- View all medicines
- Update medicine
- Delete medicine

## Technologies Used
- Java
- OOP concepts
- Collections
- DAO & DTO pattern

## How to Run
1. Clone the repository
2. Open in VS Code or Eclipse
3. Run `PharmacyMain.java`
